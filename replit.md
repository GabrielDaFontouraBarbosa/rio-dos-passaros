# Rio dos Pássaros Website

## Overview
A static educational website about Brazilian birds (Rio dos Pássaros - "River of Birds"). The site features information about various birds, a photo gallery, video carousel, quiz game, and contact form.

## Project Structure
- `/` - Main HTML files (index.html, login.html, admin.html)
- `/css/` - Stylesheets for various components
- `/js/` - JavaScript files for interactivity
- `/images/` - Image assets including bird photos, banners, and icons
- `/BOTÕES SITE/` - Bird button icons
- `/icons/` - Favicon and logo

## Technology Stack
- Static HTML/CSS/JavaScript website
- Bootstrap 5.3 for layout
- Swiper.js for carousels
- LightGallery for photo gallery
- jQuery for DOM manipulation
- Google Fonts for typography

## Running the Project
The website is served using `npx serve -s . -l 5000` on port 5000.

## Deployment
Configured as a static site deployment serving the root directory.

## Notes
- PHP contact form files exist but require a PHP server to function
- The site includes embedded YouTube videos
- ConveyThis translation widget is integrated
- Google Analytics is set up (needs measurement ID configuration)
